BerkeleyMapper 2.0 is a mapping interface for Collections (or other) Databases built on top of Google Maps. If your query results contain items with latitudes and longitudes, you may plot them using this sophisticated interface.

The Google Static Maps API lets you embed a Google Maps image on a web page (like this one) without requiring JavaScript or any dynamic page loading. The Static Maps API service creates your map based on URL parameters sent through a standard HTTP request and returns the map as an image you can display on your web page.
